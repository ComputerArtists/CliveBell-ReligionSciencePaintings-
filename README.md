README – Religion & Wissenschaft (Processing-Skript-Sammlung 2025)
Drei poetische, interaktive Visualisierungen zum Thema Glaube und Wissenschaft.
Alle Skripte laufen in Processing 4/5 (P2D-Renderer), benötigen keine externen Bibliotheken und sind sofort startklar.
1. Glaube und Gravitation
Ein goldenes/rotierendes religiöses Symbol (Kreuz, Davidstern, Halbmond, Om, Yin-Yang oder bunter Mix) zieht leuchtende Partikel an, die wie wissenschaftliche Erkenntnisse darum kreisen.

Interaktion
Mausklick → nächstes Symbol
Taste „s“ → manuell speichern
Automatisches Speichern alle ~10 Sekunden (wenn das Bild besonders schön ist)

Optik: weiche ADD-Trails, Tag-Nacht-Zyklus, hypnotisch und ausstellungstauglich.

2. Evolution des Glaubens – Weltenbaum
Ein heiliger Baum wächst langsam vor deinen Augen. An seinen äußersten Enden blühen acht große religiöse Symbole wie leuchtende Früchte.

Interaktion
Leertaste oder Mausklick → neuer Baum
Taste „s“ → Bild speichern
Taste „c“ → Farbthema wechseln (klassisch / neon / pastell / sepia)

Optik: perfekt ausbalancierte Größe, sehr luftig, meditative Stimmung – ideal als ruhiges Desktop-Kunstwerk oder Druck.

3. Kosmische Versöhnung
Zwei Universen (Wissenschaft links, Religion rechts) bewegen sich aufeinander zu. Bei Berührung entsteht ein pulsierender Moment der Versöhnung mit dem Schriftzug „VERSÖHNUNG“.

Interaktion
Leertaste oder Mausklick → neuer Zyklus
Taste „s“ → Screenshot im perfekten Moment

Optik: Planeten, Sterne, Engel, Lichtwesen und kleine religiöse Symbole tanzen sanft, bis sie sich vereinen.

Bedienung (gilt für alle drei Skripte)

s → speichert als PNG im Sketch-Ordner
Leertaste / Mausklick → meist neuer Zyklus/Baum
c → (falls vorhanden) Farbthema wechseln

Einfach die jeweilige .pde-Datei öffnen und laufen lassen – fertig.
